# PT Trisna Wiratama - Company Profile Website

Website profil perusahaan PT Trisna Wiratama, distributor alat kesehatan, laboratory supply, dan general trade.

## Teknologi

Project ini dibangun menggunakan:

*   **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
*   **Language:** [TypeScript](https://www.typescriptlang.org/)
*   **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
*   **Icons:** [Lucide React](https://lucide.dev/)

## Fitur Utama

*   **Responsive Design:** Tampilan mobile-first yang adaptif.
*   **SEO Optimized:** Struktur semantik, meta tags dinamis, dan schema markup.
*   **Product Catalog:** Katalog produk dengan filter kategori interaktif.
*   **Fast Performance:** Menggunakan Next.js Static Optimization.

## Cara Menjalankan Project

1.  **Clone Repository**
    ```bash
    git clone https://github.com/AbdisrZ/TWEB.git
    cd TWEB
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Jalankan Development Server**
    ```bash
    npm run dev
    ```

4.  Buka [http://localhost:3000](http://localhost:3000) di browser Anda.

## Struktur Project

*   `src/app`: Halaman-halaman utama (Home, About, Products, Contact).
*   `src/components`: Komponen UI yang dapat digunakan kembali (Navbar, Footer).
*   `public`: Aset statis (gambar, icon).
